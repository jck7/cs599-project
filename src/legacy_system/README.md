# 传统 OA 报销管理系统 v2.0

## 简介

纯人工传统企业报销管理系统，作为 Agent 智能化改造前的基准系统。完整实现四级审批流、预算管控、费用标准管理等企业级功能，**不含任何 AI 功能**。

## 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | Python 3 + Flask |
| 前端 | HTML + Tailwind CSS (CDN) + ECharts (CDN) |
| 数据 | Python 内存数据，重启重置 |
| 依赖 | 仅需 `pip install flask` |

## 启动

```bash
pip install flask
cd src/legacy_system
python app.py
# 浏览器访问 http://127.0.0.1:5000
```

## 核心功能

- **四级审批流**：按金额自动路由（0-1000 / 1000-5000 / 5000+）
- **费用标准**：8 级职级对应差旅/餐饮/交通上限
- **预算管控**：10 部门 × 3 年度预算，通过扣减/驳回返还
- **全链路留痕**：完整审批时间轴，操作人/时间/意见全记录
- **批量审批**：勾选多条一键通过
- **ECharts 图表**：报销趋势折线图 + 预算使用率柱状图

## 内置数据

- 10 个部门、55 名员工
- 100+ 条报销单据（覆盖草稿/审批中/已通过/已驳回/已打款/已归档）
- 完整部门年度预算数据
